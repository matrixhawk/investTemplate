## 📋 项目简介

本模板是港股/A股个股投资的**深度体检工具**，整合四大投资流派（纯硬收息/价值发现/烟蒂股/关联方资源型），专注于**安全边际、真实估值、管理人诚信**三大核心。

**核心原则**：
- 一级市场思维："10元买瓜，自己吃也值15元"
- 烟蒂股哲学：赚价值修复的钱，不赚成长的钱
- 保守主义：宁可错过，不可做错

## 🚀 快速开始（5分钟初筛）

分析任何标的前，先执行 **5分钟快速初筛**：

| 检查项 | 排除标准 | 你的标的 |
|--------|---------|----------|
| **① 价格位置** | 距52周高点<30%（已反弹>70%） | □ |
| **② 负债扫描** | 财务费用>5亿（高有息负债） | □ |
| **③ 现金覆盖** | 现金<市值20%（安全边际不足） | □ |
| **④ 盈利连续性** | 连续2年经营现金流为负 | □ |

**快速通过信号**：
- 🏆 财务费用<0 + 现金>市值50% → 无有息负债企业
- 🐢🍊 股价距52周低点<10% + T0覆盖率>80% → 标准烟蒂股

**不通过？** → 放入观察列表，等待改善

**通过？** → 打开 `个股分析标准模版.md` 进行13章深度分析

## 🎯 四流派投资体系

| 流派 | 核心特征 | 适用标的 | 分析重点 |
|------|---------|---------|----------|
| **🛡️ 纯硬收息** | 高股息（>6%）、稳定现金流 | 电信/公用事业/REITs | 派息可持续性 |
| **🔍 价值发现** | 低估（PE<10）、轻资产、FCF充裕 | 消费/制造/服务 | 真实盈利能力 |
| **🚬 烟蒂股** | 净现金>市值、靠近历史低点 | 港股小盘/国企 | 清算价值、管理人诚信 |
| **🔗 关联方资源** | 国企背景、母公司支持 | 物业/基建/公用事业 | 利益输送风险 |

## 📊 V5.4.0 核心升级（相比V5.3.4）

### 新增7大模块

| 模块 | 功能 | 实战价值 |
|------|------|----------|
| **1.3 快速初筛** | 5分钟排除法 | 提高效率，避免在垃圾标的上浪费时间 |
| **第三章-X 管理人诚信** | 历史行为审查+评分卡 | 避免汇贤类稀释陷阱 |
| **第四章-X 非现金还原** | FCF还原、识别"做亏实赚" | 真实估值（汇贤FCF 8亿 vs 报表亏损）|
| **第十三章 高级烟蒂股** | 类型判定/T级分层/REITs估值 | 天津发展/汇贤专属框架 |
| **3.3 无有息负债快速识别** | 30秒判断法 | 保利物业式极速筛选 |
| **3.5 负债结构可视化** | 对比案例图 | 直观展示风险 |
| **3.6 行业负债速查表** | 行业特征对照 | 快速定位正常/异常 |

### 关键改进

- **管理人评估**：5分制评分，识别供股/配股/稀释历史
- **资产T级分层**：T0(现金)/T1(上市股权)/T2(经营资产)/T3(低效资产)
- **REITs期限结构**：20年超级年金模型，IRR 15-18%测算
- **债务偿还路径**：明确"熬"多久（汇贤：2028年清债）
- **仓位硬约束**：绝对金额上限（汇贤≤10万港币）
