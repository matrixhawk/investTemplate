### 关键计算

```
取价 5.12元
市值 = 5.12 × 144.42 ≈ 739亿

【表面PE】
静态PE = 739 / 29.46 ≈ 25.1倍 ❌
TTM净利 = 29.46 − 11.35 + 17.90 = 36.01亿
TTM PE = 739 / 36.01 ≈ 20.5倍 ❌

【归一化（剔除一次性减值）】
2025若无21.52亿减值，利润量级回到约50亿附近
（公司表述：剔除减值后主业利润与上年基本持平）
归一化PE ≈ 739 / 51 ≈ 14.5倍 ⚠️ 贴15倍

【FCF双口径】
报告口径FCF ≈ OCF 72.09 − CAPEX（轻资产，数亿级）
  → FCF中枢约65–70亿
  FCF/市值 ≈ 9%
  FCF倍数 ≈ 10.5–11.4倍 ✅ 显著好于PE叙事
维护性CAPEX极低（点位运营为主），维护性FCF ≈ 报告FCF
置信度：A（OCF明确，CAPEX量级小）

【股息】
0.19元/股 / 5.12 ≈ 3.7%（按10派1.9）
若按100%派息于29.46亿：仍仅约3.7–4%
归一化50亿×80%派息假设：股息率可至约5%+

【净现金】
历史现金数十亿、有息负债极低；
2025末净资产约146亿，货币资金随理财配置波动。
净现金/市值有限（约10%量级历史），不构成烟蒂。
```

## 第三章：经营分析

### 3.1 2025拆解

- 楼宇媒体收入约120.3亿（+4.2%），毛利率提升（点位成本优化）  
- 互联网客户收入大增（+188%级），结构改善  
- **利润暴雷主因**：数禾科技长期股权投资减值21.52亿  
- 扣非仍-42%：除减值外，联营投资与基数亦有影响；主业经营层表述为持平量级  

### 3.2 2026Q1

- 收入微增，利润+58%  
- 含退出数禾相关非经常收益（资本公积转回、递延税等）→ **Q1利润质量需打折**  
- 不可将17.9亿×4简单年化  

### 3.3 同业

梯媒格局集中；竞争在新潮媒体等，分众点位与品牌仍领先。

## 第七章：决策

### 支持事实
1. 梯媒龙头点位壁垒，毛利率高  
2. OCF/FCF持续强，轻资产  
3. 减值后主业利润有望回归50亿级  

### 证伪
1. 主业楼宇收入连续两季双位数下滑  
2. 再曝大额非主业减值  
3. OCF/收入比显著恶化  

### 最重要变量
**广告主预算周期 + 主业扣非利润能否站稳50亿**
