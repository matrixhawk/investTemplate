## 步骤0：框架选择（修正）

### 0.1 市场在交易什么？

| 逻辑 | 权重 | 说明 |
|------|------|------|
| **白酒行业周期底部** | 高 | 政商务场景收缩、消费降级、价盘承压 |
| **渠道去库存/会计出清** | 高 | 2025收入确认趋严 = 报表“挤水分” |
| **历史估值/价格低位** | 高 | 股价多年低位，风险偏好极低 |
| 品牌长期价值 | 中 | 浓香龙头心智仍在，但要便宜才买 |
| 表面TTM PE | **低（失真）** | 不能机械套用稳定收息股的PE<10/15 |

### 0.2 决策树

```
拿到五粮液
    ↓
重大事件催化剂（重组/定增）？ → 否
    ↓
利润大幅下滑/行业调整 + 股价历史低位？
    → 是 → 🟠 周期拐点 / 去库存修复（主框架）
    ↓
兼有强品牌、厚现金？ → 🟡 价值回归（次框架）
    ↓
核心问题不是“PE是否<10”，而是：
    ① 周期走到哪了？（价盘、库存、动销）
    ② 经销商报表外的真实库存是否出清？
    ③ 中周期利润中枢是多少？
```

**最终判定**：🟠 **周期拐点（白酒弱周期）** 为主，🟡 价值回归为辅。  
**修正前错误**：用稳定盈利公司的TTM PE硬约束一票否决历史低位的周期出清标的——工具用错了（类似山东药玻选错框架的教训，方向相反但是同一类错误）。

## 第三章：周期与渠道——核心章节 ⭐

### 3.1 为什么白酒必须谈周期

高端白酒需求绑定：
1. 商务宴请与礼赠  
2. 居民资产负债表与风险偏好  
3. **经销商加杠杆囤货**的库存周期（往往比终端需求波动更大）

因此报表收入有两层：
- **真实动销**（消费者开瓶）  
- **渠道吞吐**（厂家→一批→二批/终端的压货与去化）

> **你不知道经销商是否囤货** = 你不知道当前利润里有多少是“未来的需求被提前确认/或回补库存”。  
> 这是白酒分析的第一性原理，优先于PE表格。

### 3.2 2025会计调整在渠道语境下的含义

| 现象 | 渠道解读 |
|------|----------|
| 2025营收/净利大幅下修 | 收入确认更谨慎 = 报表层“减少提前确认” |
| 其他流动负债/合同负债抬升（点评口径） | 未确认收入趴在负债端，**不像**简单“卖不动清空”那么干净 |
| 2026Q1利润爆炸式反弹 | **两种可能**：①真实旺季动销+份额；②低基数+渠道回补/确认节奏变化 |

**不能**把Q1 +82%直接年化成“重回300亿净利时代”。

### 3.3 经销商囤货：如何观测（可跟踪清单）

| 观测指标 | 好信号（去库存完成/真实动销） | 坏信号（仍在囤货或二次压货） | 数据来源等级 |
|----------|-------------------------------|-------------------------------|--------------|
| **批价** | 稳中有升或横盘不破关键价 | 持续阴跌、倒挂加深 | B/C（渠道调研） |
| **开箱率/动销** | 开箱率维持高位（公司曾披露Q1开箱率等改善表述） | 开箱率掉、终端不动厂家出货 | B（公司交流/研报） |
| **合同负债/预收款** | 与动销同步、结构健康 | 预收暴增但批价跌 = 压货嫌疑 | S/A（财报） |
| **经营现金流 vs 净利** | OCF≥净利 | 净利大增OCF跟不上 | S |
| **经销商打款意愿** | 主动打款、要货 | 靠政策压货、搭赠才能出货 | B/C |
| **库存天数（一批）** | 降至历史中枢以下 | 仍高于正常月销数倍 | C（难精确） |
| **控货挺价执行** | 厂家限量、价盘稳 | 明面挺价暗地冲量 | B |

### 3.4 两种情景（仓位含义不同）

```
情景A：渠道已出清，Q1是真实修复
  → 中周期净利中枢向 220–280 亿靠拢
  → 现价对应约 10–13x，历史低位+厚现金 → 可持有/网格加仓

情景B：渠道仍在补库存或确认节奏制造低基数反弹
  → 中周期净利中枢停在 150–200 亿
  → 现价对应 14–19x，且还有二次去库存风险
  → 试探仓止损，不加仓
```

**当前先验**：A、B都有可能；价格已部分反映B，故允许**不对称小仓**，但必须用上表跟踪切换概率。

### 3.5 利润归因（简表）

| 项目 | 2024 | 2025调整后 | 读法 |
|------|------|------------|------|
| 营收 | 892亿 | 405亿 | 口径+行业双杀 |
| 归母净利 | 319亿 | 90亿 | 不可直接同比决策 |
| 毛利率 | 高位稳定 | 仍高（约82%+） | 品牌定价权未崩 |
| 2026Q1净利 | — | 81亿 | 弹性大，真假待验 |

同业：茅台韧性更强、估值体系不同；五粮液弹性更大（涨跌都更猛），周期位置更敏感。

## 第七章：决策（修正后）

### 7.1 三条支持事实

1. **价格已处多年低位**，风险偏好与估值泡沫双重压缩  
2. **零有息负债 + 巨额现金 + 地方国企品牌**，熊市生存能力极强  
3. 2025报表挤水分后，若渠道出清，向上弹性来自利润中枢修复而非泡沫扩张  

### 7.2 三条证伪条件（渠道导向）

1. **批价**较关键支撑位再明显破位，且持续无稳  
2. 中报/年报 **合同负债与出货** 显示压货，同时 **OCF 显著弱于利润**  
3. 公司或渠道反馈 **开箱率/动销** 走弱，而厂家仍强压任务  

### 7.3 当前最重要变量

> **经销商库存是否真实出清，以及终端动销是否跟得出货。**  
> （不是“PE多少”，也不是“会不会破产”。）

### 7.4 操作计划

| 阶段 | 条件 | 动作 |
|------|------|------|
| **试探建仓** | 现价历史低位区（约70–78）+ 现金安全垫 | **3–5%** 分批（如 2%+2%） |
| **持有观察** | 批价平稳、开箱/动销不差、OCF匹配 | 持有，不追高加杠杆 |
| **网格加仓** | 批价稳 + 中报验证去库存 + 中周期利润上修 | 加至 **6–8%** |
| **停止加仓** | 仅股价涨、渠道指标恶化 | 机械遵守，不加 |
| **减仓/退出** | 证伪条件触发 | 先减半再清仓 |

| 价格带（粗算） | 含义 |
|----------------|------|
| ≤70 | 更悲观，可偏试探上限（仍≤5%起步） |
| 70–78 | **当前试探区** |
| 78–90 | 需渠道验证才能加 |
| >100 | 周期修复已部分计价，换手或降预期 |

### 7.5 仓位与纪律

- 建议上限：**试探 3–5%**（承认周期误判风险）  
- 渠道验证后：**≤8%**  
- 禁止：因“历史新低”单边重仓；禁止忽视囤货变量
