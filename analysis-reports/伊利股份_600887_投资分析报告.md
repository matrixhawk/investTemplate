### 关键指标计算过程

```
取基准价 26.80元（2026-07中下旬区间中枢）
市值 = 26.80 × 63.25 ≈ 1,695亿人民币

【表面PE — 交叉用，不作乳业唯一结论】
静态PE = 1,695 / 115.65 ≈ 14.66倍 ⚠️

TTM净利润 = 115.65 − 48.74 + 53.95 = 120.86亿
TTM PE = 1,695 / 120.86 ≈ 14.03倍 ⚠️（10–15倍书面例外区）

【乳业强制：FCF主估值】
报告口径FCF = 经营现金流 − 现金流量表资本开支
  OCF ≈ 143.4亿
  CAPEX：年报投资活动含购建固定资产等，综合多源估算全年购建类约20–30亿
  报告口径FCF区间 ≈ 113–123亿（中枢约118亿）
  置信度：B（OCF为S/A，CAPEX为估算区间）

维护性FCF：
  乳业维持性CAPEX（产线维护/冷链）通常低于扩张期；
  若维护性CAPEX取总CAPEX的60%–80% → 约12–24亿
  维护性FCF ≈ 119–131亿（中枢约125亿）
  置信度：C

FCF/市值（中枢118）≈ 7.0%
维护性FCF/市值（中枢125）≈ 7.4%
FCF倍数 ≈ 1,695/118 ≈ 14.4倍 ✅ 优于表面PE叙事

【股息】
分红总额87.29亿 / 市值1,695 ≈ 5.15%
每股约1.38元 / 26.80 ≈ 5.15%
公司承诺2025–27分红率不低于75%

【净现金】
货币资金约177亿级；有息负债（短借+长借等）乳业通常不低
雪球等讨论提及有息负债可至数百亿量级、资产负债率约60%
→ 净现金可能为负或很低，不构成安全垫，禁止“剔除净现金低估”叙事
```

## 步骤0：框架选择

- 无重组/定增主线事件

- **非**烟蒂（PB不低）、**非**成长股零派息

## 第三章：经营与利润归因

### 3.1 2025业绩要点

- 营收1,156亿，同比约+0.2%（存量博弈）
- 归母净利115.65亿，+36.8%（利润率扩张）
- 液奶收入约704亿，-6.1%（量价压力）
- 原奶成本红利 + 费用管控 → 净利率升至约10%
- 2026Q1：营收约347亿（+5.5%级），净利54亿（+10.7%），液奶转正增长约2.3%

### 3.2 利润质量

- 扣非净利增速更高（费用与减值低基数效应）
- OCF 143亿 > 净利，**利润含金量优于2024困难年份**
- 风险：利润高增部分来自成本红利，原奶价格回升可能压缩利润率

### 3.3 同业（vs 蒙牛）

| 维度 | 伊利 | 蒙牛（港股） |
|------|------|--------------|
| 市场 | A股 | 港股 |
| 2025利润趋势 | 强修复 | 仍承压/分化 |
| 估值框架 | FCF+股息 | FCF（模板既有案例） |
| 分红 | 高分红承诺 | 相对克制/波动 |

## 第四章：现金流

| 项目 | 2025 | 说明 |
|------|------|------|

**疑点**：资产负债率偏高、有息负债规模 — 必须每季核对短债与现金流匹配，防止“高分红+高杠杆”组合恶化。

## 第六章：估值（FCF优先）

| 指标 | 数值 | 判定 |
|------|------|------|
| 静态PE | 14.7x | 交叉，非结论 |
| TTM PE | 14.0x | 书面例外区 |
| FCF收益率 | ~7.0% | ✅ 收息框架下有吸引力 |
| 股息率 | ~5.2% | ✅ 达标（>4–5%） |
| 净现金故事 | 不适用 | 杠杆不低 |

**书面例外理由（PE 14x）**：
1. 乳业PE失真历史教训（蒙牛案例）— 以FCF为准  
2. FCF/市值约7% + 股息约5% + 分红承诺  
3. 2026Q1液奶转正，成本红利仍在但经营有企稳信号  

**不满足**：TTM PE<10 的无脑建仓标准；**正式底仓需更低估值或FCF进一步抬升**。

## 第七章：决策

### 支持事实
1. 龙头份额 + 2025利润与净现比显著修复  
2. 分红率≥75%承诺至2027，股息率约5%  
3. FCF收益率约7%，优于单纯看PE  

### 证伪条件
1. 连续两季经营现金流同比大幅转负或净现比<0.7  
2. 取消/实质下调分红承诺  
3. 液奶收入连续三年下滑且无结构升级对冲  

### 最重要变量
**原奶成本趋势 + 液奶量价能否稳住**
