# 纳斯达克100 · VIX逆向轮动定投策略

> **一页纸执行卡** · 适用：2015至今历史验证 · 等额本金 · 只在股/现金间轮动 · 不卖飞不深套

---

## 一、核心一句话逻辑

**VIX低 = 市场狂热 → 卖部分持仓换现金  
VIX高 = 市场恐慌 → 现金全买回  
每月按VIX定投，高位少买、低位多买  
靠"高卖低买"扩大份额、提高收益、降低回撤**

---

## 二、每月定投买入规则（每月1号执行）

**基准**：每月目标投入 **$1000**，总本金严格不变

| VIX 区间 | 市场状态 | 当月买入金额 | 说明 |
|---------|---------|-------------|------|
| ＜13 | 极度狂热 | $300 | 少投$700存入现金池 |
| 13～20 | 乐观高位 | $700 | 少投$300存入现金池 |
| 20～30 | 正常震荡 | $1000 | 基准投入 |
| 30～45 | 明显恐慌 | $2000 | 从现金池取$1000加仓 |
| 45～60 | 深度大跌 | $3500 | 从现金池取$2500加仓 |
| ≥60 | 史诗级崩盘 | $6000 | 从现金池取$5000加仓 |

> **关键**：少投的钱自动存为"现金池"，恐慌月从池子里加仓，不额外加钱。

---

## 三、卖出规则（只看VIX，不猜顶）

**触发条件**：VIX **连续 2 个月 ＜ 13**  
→ 代表纳指处于**连续高位、全民乐观**

**操作**：卖出总持仓的 **50%**，全部转为现金  
现金不转出、不消费，留在账户等待大跌。

---

## 四、买回规则（只看VIX，不抄底猜底）

**触发条件**：VIX **≥ 30**  
→ 代表市场进入恐慌、指数大幅回调

**操作**：手中**所有现金，一次性全额买回**纳斯达克100  
把高位卖出的筹码，在低位加倍接回。

---

## 五、全年完整执行流程（每月只做3步）

```
每月1号：
  ├─ 1. 查VIX → 确定定投金额（按第二节表格）
  ├─ 2. 检查是否连续2个月VIX＜13
  │     ├─ 是 → 卖出50%持仓，转现金
  │     └─ 否 → 继续持有
  └─ 3. 检查VIX是否≥30
        ├─ 是 → 现金全部买回
        └─ 否 → 继续等待
```

## 六、策略最终效果（2015–至今）

| 指标 | VIX逆向策略 | 普通定投 | 一次性投入 |
|------|-----------|---------|-----------|
| **总收益** | 高于两者 | 中等 | 看入场时点 |
| **回撤控制** | 最小 | 中等 | 最大 |
| **资金效率** | 最高（低买高卖） | 平均 | 一次性占用 |
| **心理压力** | 需承受恐慌期重仓 | 恒定 | 需承受波动 |

**核心优势**：完全逆向，不预测、不主观、只看情绪指标执行。

## 八、风险提示

1. **现金流压力**：VIX≥30时需要动用现金池，极端情况下可能需要动用$6000/月，确保资金链充足
2. **VIX与纳指相关性变化**：历史规律不代表未来，VIX可能失效
3. **踏空风险**：VIX＜13时减仓，若市场继续上涨会跑输死拿
4. **本策略未考虑**：交易成本、税费、汇率（非美元资金）、滑点
